# Pre-req

Install Xcode

``` xcode-select --install ```

Install brew 

``` /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" ```


# Clone the Repo
 ```
 git clone https://github.com/mohanvamc/sre-mac-setup.git

 ```

# Run the script
```
cd sre-mac-setup
chmod +x sre-tools-mac.sh
./sre-tools-mac.sh

```
